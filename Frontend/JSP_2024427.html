¿Qué es JSP (JavaServer Pages)?

JSP (JavaServer Pages) es una tecnología de Java que permite crear páginas web dinámicas en el lado del servidor. Fue desarrollada por Sun Microsystems y es ampliamente utilizada en aplicaciones web empresariales.

¿Cómo funciona JSP?

Las páginas JSP contienen código HTML combinado con fragmentos de código Java. Cuando un usuario solicita una página JSP, el servidor la convierte en un servlet Java, lo compila y ejecuta, generando así el HTML final que se envía al navegador.

¿Para qué sirve JSP?

  • Crear páginas web dinámicas e interactivas.
  • Separar la lógica de negocio del diseño web.
  • Integrar aplicaciones Java con la web.
  • Facilitar el desarrollo de aplicaciones empresariales.

Ventajas de JSP

  • Permite reutilizar componentes Java (servlets, beans, etc.).
  • Fácil integración con bases de datos.
  • Separación entre contenido y lógica.
  • Soporte para plantillas y bibliotecas de etiquetas.

Ejemplo básico de código JSP

<html>
<body>
    <h2>Hola, <%= request.getParameter("nombre") %>!</h2>
</body>
</html>

Paso a paso: Cómo agregar JSP a un proyecto Maven en NetBeans

  1. Crear un nuevo proyecto Maven Web:
    • En NetBeans, ve a Archivo > Nuevo Proyecto.
    • Selecciona Java With Maven > Aplicación Web y haz clic en Siguiente.
    • Asigna un nombre y una ubicación al proyecto y haz clic es (es opcional borra el nombre del paquete) Finalizar.
  2. Verifica la estructura del proyecto:
    • Debes tener una carpeta src/main/webapp donde irán tus archivos JSP.
  3. Agrega un archivo JSP:
    • Haz clic derecho en webapp (nombre del proyecto) > Nuevo > JSP.
    • Asigna un nombre (por ejemplo, index.jsp) y haz clic en Finalizar.
  4. Configura el servidor:
    • Haz clic derecho en el proyecto > Propiedades > Run.
    • Selecciona un servidor compatible (GlassFish Server).
  5. Agrega dependencias si es necesario:
    • Para proyectos Maven, normalmente no necesitas agregar dependencias para JSP, pero si usas MySQL, agrega en tu pom.xml:
    <dependency>
    <groupId>mysql</groupId>
    <artifactId>mysql-connector-java</artifactId>
    <version>8.0.33</version>
</version>
</dependency>
  6. Ejecuta el proyecto:
    • Haz clic derecho en el proyecto y selecciona Ejecutar o presiona F6.
    • El navegador abrirá tu archivo index.jsp o index.html.
