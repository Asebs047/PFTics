Tutorial HTML

¿Qué es HTML?

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript). "Hipertexto" hace referencia a los enlaces que conectan páginas web entre sí, ya sea dentro de un único sitio web o entre sitios web. Los enlaces son un aspecto fundamental de la Web. Al subir contenido a Internet y vincularlo a las páginas creadas por otras personas, te conviertes en un participante activo en la «World Wide Web» (Red Informática Mundial).

¿Por qué aprender HTML?

Aprender HTML es fundamental para cualquier persona interesada en el desarrollo web. Es el punto de partida para crear páginas web y aplicaciones, permitiendo estructurar el contenido y facilitar la integración con otras tecnologías como CSS y JavaScript.

  • Es fácil de aprender y entender.
  • Permite crear sitios web desde cero.
  • Es compatible con todos los navegadores modernos.
  • Es la base para aprender otros lenguajes y frameworks web.

¿Dónde se usa HTML?

HTML se utiliza en:

  • Páginas web personales y profesionales.
  • Aplicaciones web y móviles.
  • Correos electrónicos con formato.
  • Documentación digital y presentaciones interactivas.

Un ejemplo simple de HTML 


<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejemplo de Código</title>
</head>
<body>
    <h1>¡Hola Mundo!</h1>
    </body>
</html>

Ejemplo explicado

  • La declaración "!DOCTYPE html" define que este documento es un documento HTML5
  • El elemento "html /html" es el elemento raíz de un HTML página
  • El elemento "head /head" contiene metainformación sobre el Página HTML
  • El elemento "title /title" especifica un título para el Página HTML (que se muestra en la barra de título del navegador o en la pestaña de la página)
  • El elemento "body /body" define el elemento document, y es un contenedor para todo el contenido visible, como encabezados, párrafos, imágenes, hipervínculos, tablas, listas, etc.
  • El elemento "h1 /h1" define un encabezado grande

    Ejemplos de etiquetas HTML comunes

    <p>: Se usa para definir párrafos de texto en HTML.

    
<p>Aquí puedes escribir lo que quieras.</p>

    <h1> a <h6>: Definen títulos y subtítulos con diferentes niveles de importancia.

    
<h1>Encabezado principal</h1>
<h2>Subtítulo</h2>
<h3>Subtítulo de tercer nivel</h3>
<h4>Subtítulo de cuarto nivel</h4>
<h5>Subtítulo de quinto nivel</h5>
<h6>Subtítulo de sexto nivel</h6>

    Encabezado principal

    Subtítulo

    Subtítulo de tercer nivel

    Subtítulo de cuarto nivel

    Subtítulo de quinto nivel
    Subtítulo de sexto nivel

    <a>: Permite crear hipervínculos a otras páginas o recursos.

    
<a href="https://www.ejemplo.com">Visitar Ejemplo</a>
    Visitar Ejemplo

    <img>: Permite mostrar imágenes en la página web.

    
<img src="imagen.jpg" alt="Descripción de la imagen">

    <ul> y <ol>: Listas ordenadas y desordenadas.

    
<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ul>

<ol>
  <li>Primer elemento</li>
  <li>Segundo elemento</li>
</ol>
    • Elemento 1
    • Elemento 2
    1. Primer elemento
    2. Segundo elemento

    <table>: Permite organizar datos en filas y columnas.

    
<table>
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
  </tr>
  <tr>
    <td>Juan</td>
    <td>25</td>
  </tr>
</table>
    Nombre Edad
    Juan 25

    <form>: Permite recopilar datos del usuario mediante campos de entrada.

    
<form>
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre">
  <input type="submit" value="Enviar">
</form>

    Ejemplos prácticos de etiquetas HTML

    <button>: Para crear botones interactivos.

    
<button>Haz clic aquí</button>

    <div> y <span>: Para agrupar y estilizar contenido.

    
<div>Este es un bloque de contenido</div>
<span>Este es un texto en línea</span>
    Este es un bloque de contenido
    Este es un texto en línea

    <input>: Para diferentes tipos de campos en formularios.

    
<input type="text" placeholder="Escribe tu nombre">
<input type="checkbox"> Acepto los términos
<input type="radio" name="opcion" value="1"> Opción 1

    Acepto los términos
    Opción 1

    <select> y <option>: Para menús desplegables.

    
<select>
  <option value="1">Opción 1</option>
  <option value="2">Opción 2</option>
</select>

    <textarea>: Para áreas de texto extensas.

    
<textarea rows="4" cols="30">Escribe aquí...</textarea>

    <iframe>: Para incrustar contenido externo.

    
<iframe src="https://www.ejemplo.com" width="300" height="200"></iframe>

    Conclusión

    HTML es la base fundamental para crear y estructurar páginas web. Aprenderlo te permitirá comprender cómo funciona la web y te abrirá las puertas para explorar otras tecnologías como CSS y JavaScript. Dominar HTML es el primer paso para convertirte en desarrollador web y crear tus propios proyectos digitales.
